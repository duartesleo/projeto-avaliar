Aplicação para aprendizagem do NodeJs.

Essa aplicação tem como objetivo avaliar serviços/produtos de empresas,
atraves de um sistema simples e rapido, mantendo historico e revelando para
o usuario um grafico para tomada de decisão.

-- Requisitos funcionais

[ ] Deve ser possivel cadastrar um usuario
[ ] Deve ser possivel cadastrar um serviço/produto
[ ] Deve ser possivel cadastrar uma pergunta para avaliar
[ ] Deve ser possivel selecionar as perguntas que deseja enviar para o cliente responder

-- Regra de Negocio

[ ] Não deve ser possivel cadastrar um cliente duplicado
[ ] Não deve ser possivel enviar a mesma pegunta duas vezes para o mesmo cliente 
pelo mesmo serviço/produto
[ ] Não pode aceitar responstas fora de 1 a 5
